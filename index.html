aefCMS

aefCMS is a simple CMS written in Java, using the Apache Velocity template engine.

By the way, this very web page has been made with aefCMS:

WHAT

aefCMS is a Content Management System capable to generate static,
single-page, Bootstrap based websites.
It is licensed under agpl-3.0, therefore you are free to access its code
and modify it from its repository.

HOW

aefCms is built with the following technologies:

Java ZK Framework Apache Velocity Google gson Bootstrap

WHERE

aefCMS was developed during our ITS Tech Talent Factory summer internship
at Opengate S.R.L. under the supervision of Luca Visconti.
